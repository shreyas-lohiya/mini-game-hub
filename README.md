# Mini Game Hub

## Ritwik Khandelwal and Shreyas Lohiya

A secure,password protected multi-user game hub built using **Bash**,**Python**, and **Pygame**, where two authenticated players can choose from multiple board games, play via a GUI, and track results on a leaderboard.

## Features
### - User Authentication (Bash)
Secure login system with SHA-256 password hashing

New user registration support
### - Multiple Games (Pygame GUI)
Tic-Tac-Toe (10×10, 5 in a row)

Othello (Reversi)

Connect Four

### - Object-Oriented Game Engine
Base class for all 2-player board games

Each game implemented via inheritance
### - Leaderboard & Analytics
Persistent match history (history.csv)

Leaderboard with win/loss stats

Graphs using Matplotlib (top players, game distribution)
